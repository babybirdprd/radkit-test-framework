## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-init-agent`
- `allow-chat`
- `allow-stream-chat`
- `allow-submit-tool-output`
- `allow-search-memory`
- `allow-save-memory`
- `allow-delete-memory`
- `allow-list-tasks`
- `allow-get-task`
- `allow-cancel-task`
