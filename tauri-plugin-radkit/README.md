# Radkit Tauri Plugin

The official Tauri plugin for [Radkit](https://github.com/agents-sh/radkit).

**Documentation**: See the [Documentation](../plugin-docs/README.md) for installation, usage, and API reference.

## Features

*   Full Radkit Runtime support
*   Unified LLM API (OpenAI, Anthropic, etc.)
*   Frontend Tool definition and execution
*   Vector Memory
*   Streaming Support
